# NLP Projects Repository

Repositori ini berisi kumpulan proyek yang seluruhnya berfokus pada **Natural Language Processing (NLP)**. Setiap proyek di dalamnya mengeksplorasi berbagai aspek pemrosesan bahasa alami, seperti:

- Text preprocessing dan feature engineering  
- Representasi teks (embedding, vectorization)  
- Pemodelan bahasa dengan machine learning atau deep learning  
- Analisis sentimen, klasifikasi teks, atau tugas NLP lainnya  
- Pemanfaatan model bahasa besar (LLM) atau transformer  

Tujuan utama repositori ini adalah untuk mendokumentasikan eksperimen, pembelajaran, dan implementasi praktis terkait NLP yang dapat menjadi referensi untuk penelitian, pembelajaran, maupun pengembangan aplikasi berbasis bahasa.
